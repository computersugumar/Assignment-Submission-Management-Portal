# Assignment-Submission-Management-Portal
Developed a desktop-based Assignment Submission Management System using Python, Tkinter, and SQLite to manage student records efficiently. Features include secure login, data entry, search by key fields, and Excel export, ensuring organized, accurate, and automated academic record handling.
The IGNOU Assignment Submission Management System is a desktop-based application developed to simplify and automate the process of managing student assignment records in an efficient and organized manner. It eliminates the need for manual record-keeping by providing a structured digital solution that improves accuracy, reduces redundancy, and saves administrative time.

The application is built using Python, with Tkinter used to design an interactive and user-friendly graphical interface, and SQLite used as the backend database for storing and managing data. The system includes a secure login mechanism to ensure controlled access and protect sensitive information.

Users can easily add new assignment records by entering details such as enrollment number, student name, program code, course code, assignment details, and submission date. The system also provides a well-organized view section where users can access the complete dataset or perform specific searches using key fields like enrollment number, program code, or course code.

One of the major strengths of this application is its ability to handle and organize large amounts of data efficiently, making information retrieval quick and reliable. Additionally, the system supports exporting records into Excel format, allowing users to generate reports and maintain backups for future reference.

Proper input validation and error-handling mechanisms are implemented to ensure data consistency and prevent invalid entries. The project reflects strong fundamentals in GUI design, database integration, and real-world problem solving, making it a practical and scalable solution for academic institutions.

🛠️Tech Stack
  Python
  Tkinter (GUI)
  SQLite (Database)
  OpenPyXL (Excel Export)

🚀 Features
  Secure login authentication
  User-friendly GUI using Tkinter
  Add and manage student assignment records
  Search functionality (Enrollment No, Program Code, Course Code)
  View complete dataset in table format
  Export data to Excel
  Input validation and error handling
